# How to create a venv

Build a venv in py
``py -3 -m venv venv`` (Windows)
``python3 -m venv <name>`` (Mac)

Run venv
``.\venv\Scripts\Activate.ps1`` (Win)
``source venv/bin/activate`` (Mac)
